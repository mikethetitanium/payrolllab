Payroll Data Model

employees
  • id (PK)
  • employee_name
  • country
  • department
  • hire_date
payroll_runs
  • id (PK)
  • payroll_month
  • country
  • run_date
payroll_results
  • id (PK)
  • employee_id (FK → employees)
  • payroll_run_id (FK → payroll_runs)
  • gross_salary
  • tax_amount
  • pension
  • net_salary
deductions
  • id (PK)
  • employee_id (FK → employees)
  • deduction_type
  • amount

Relationships

  • employees → payroll_results: One employee has many payroll results (one per month)
  • payroll_runs → payroll_results: One payroll run contains many employee results
  • employees → deductions: One employee can have multiple deductions

Key Concepts

  • Payroll Run: Represents a payroll processing cycle for a specific country and month
  • Payroll Result: The calculated payroll for one employee in one payroll run
  • Gross Salary: Total salary before deductions
  • Net Salary: Take-home pay after tax and pension deductions
